### 📝 Tentang Proyek

**ArkaikPedia** adalah sebuah platform web **Museum Digital & Ensiklopedia** interaktif yang dirancang untuk menyajikan informasi sejarah, artefak, atau data ensiklopedia secara modern. Proyek ini dibangun menggunakan arsitektur frontend murni (*client-side*) yang efisien dengan memanfaatkan utilitas Tailwind CSS untuk desain responsif dan file JSON lokal sebagai sumber datanya.

---

### 🚀 Fitur Utama Aplikasi

- 🎨 **Desain Modern & Responsif:** Tampilan estetik dengan skema warna kustom (*stone palette*) yang sepenuhnya responsif di perangkat mobile maupun desktop menggunakan Tailwind CSS.
- 🗄️ **Data Terpusat (JSON):** Semua konten informasi museum dan ensiklopedia dikelola secara rapi di dalam satu file file data tanpa memerlukan konfigurasi database server yang rumit.
- ⚡ **Ringan & Cepat:** Menggunakan integrasi Tailwind via CDN dan struktur HTML asli sehingga waktu pemuatan halaman menjadi sangat instan.

## 💻 Panduan Cara Menjalankan Proyek

Karena proyek ini menggunakan teknologi statis (*Frontend-only*), kamu tidak memerlukan XAMPP atau Laragon untuk menjalankannya. Ikuti langkah mudah ini:

### Langkah 1: Unduh (Clone) Repositori
Buka terminal, Git Bash, atau CMD di komputer kamu, lalu jalankan perintah:
bash
git clone [https://github.com/toriqayataalmukhorobin-del/arkaikpedia.git](https://github.com/toriqayataalmukhorobin-del/arkaikpedia.git)
cd arkaikpedia


### Langkah 2: Jalankan Halaman Web
Kamu bisa membuka proyek ini dengan dua cara mudah:
 * **Cara Cepat:** Cukup masuk ke folder proyek lalu klik dua kali (*double-click*) pada file index.html untuk langsung membukanya di browser favoritmu.
 * **Rekomendasi (VS Code):** Buka folder proyek ini di VS Code, lalu gunakan ekstensi **Live Server** (klik kanan pada index.html -> pilih *Open with Live Server*) agar fitur pembaruan otomatis aktif saat kamu mengubah kode.
## 📁 Struktur Dokumen Utama
 * images/ — Folder penyimpanan aset gambar, ikon, dan ilustrasi konten museum.
 * database.json — Pusat penyimpanan data teks atau informasi ensiklopedia yang akan ditampilkan ke web.
 * index.html — Berisi struktur utama halaman web, konfigurasi kustom Tailwind CSS, serta logika penampil data.
